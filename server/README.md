# Fish Cart - Server

Backend API server for Fish Cart application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key_here
```

3. Start server:
```bash
# Development mode
npm run dev

# Production mode
npm start
```
